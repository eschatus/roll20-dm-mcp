# roll20-dm-mcp

AI-assisted D&D 5e session management for **Roll20 + D&D Beyond**. The DM speaks; the gem acts.

## Guides (you don't need to clone the repo)

- **[Setup](Setup)** — install, configure, and first run.
- **[Voice HUD Gem](Voice-HUD-Gem)** — the push-to-talk scrying-gem overlay: tabs, inbox, and controls.
- **[Player Commands](Player-Commands)** — `!dm`, `!tactics`, `!recall`, `!rules`: what your *players* can type in Roll20 chat.

## Developer docs (in the repo)

The deep-dive material — architecture, the Roll20 RT / D&D Beyond protocols, design decisions,
security posture, API coverage — lives **in the repository** under
[`docs/`](https://github.com/eschatus/roll20-dm-mcp/tree/master/docs), because it versions with the
code and would drift if copied here. Start with the repo
[README](https://github.com/eschatus/roll20-dm-mcp#readme) and `CLAUDE.md`.
